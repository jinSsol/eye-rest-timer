# PRD: Eye Rest Timer (눈 휴식 타이머)

## 1. 개요

### 1.1 제품 비전
디지털 기기 사용으로 인한 눈 피로를 예방하고, 건강한 눈 습관을 형성할 수 있도록 돕는 타이머 앱

### 1.2 목표 사용자
- 장시간 컴퓨터/스마트폰 사용자
- 재택근무자, 학생, 개발자
- 눈 건강에 관심 있는 모든 연령대

### 1.3 핵심 가치
- **심플함**: 복잡한 설정 없이 바로 사용
- **효과성**: 과학적으로 검증된 20-20-20 규칙 적용
- **지속성**: 통계와 연속 기록으로 동기부여

## 3. 비기능 요구사항

### 3.1 성능
- 앱 시작 시간: 2초 이내
- 타이머 정확도: ±1초
- 배터리 소모: 최소화 (백그라운드 최적화)

### 3.2 안정성
- 앱 크래시율: 0.1% 미만
- 백그라운드 알림 안정성: 99%

### 3.3 호환성
- Android: 6.0 (API 23) 이상
- iOS: 12.0 이상

## 4. 광고 전략

### 4.1 광고 배치
| 위치 | 광고 유형 | 노출 조건 |
|-----|---------|----------|
| 메인 화면 하단 | 배너 광고 | 상시 |
| 휴식 완료 후 | 전면 광고 | 3회 휴식마다 |
| 설정 나갈 때 | 전면 광고 | 설정 변경 후 |

### 4.2 광고 정책 준수
- 실수 클릭 유도 금지
- 콘텐츠와 광고 명확히 구분
- 전면 광고 빈도 제한 (5분에 1회 이하)
