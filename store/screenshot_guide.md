## 권장 스크린샷 구성 (5~6장)

### 1. 타이머 화면 - 시작 전
**캡션:** "눈 건강을 위한 첫 걸음"
- idle 상태의 타이머 화면
- 귀여운 눈 캐릭터가 보이도록
- "시작하기" 버튼 강조

### 2. 타이머 화면 - 진행 중
**캡션:** "예쁘게 채워지는 진행률"
- 50% 정도 진행된 상태
- 배경색이 채워진 모습
- 응원 메시지가 보이도록

### 3. 휴식 화면
**캡션:** "20초 휴식, 눈이 편해져요"
- 휴식 오버레이가 표시된 상태
- 카운트다운 숫자가 보이도록

### 4. 기록 화면
**캡션:** "나의 눈 건강 기록"
- 오늘 휴식 횟수
- 연속 달성 일수
- 주간 그래프

### 5. 설정 화면
**캡션:** "나에게 맞게 설정"
- 시간 설정 슬라이더
- 테마 선택 옵션

### 6. 온보딩 화면 (선택)
**캡션:** "20-20-20 규칙"
- 첫 번째 온보딩 페이지
- 앱의 핵심 가치 전달

## 스크린샷 촬영 팁

1. **시뮬레이터 사용**
   ```bash
   # iOS 시뮬레이터
   xcrun simctl io booted screenshot screenshot.png

   # 또는 Cmd + S (시뮬레이터에서)
   ```

2. **상태바 정리**
   - 시간: 9:41 (Apple 공식 시간)
   - 배터리: 100%
   - 신호: 풀 바

3. **다크모드 버전**
   - 라이트/다크 모드 모두 준비하면 좋음

4. **텍스트 오버레이**
   - Figma, Canva 등으로 캡션 추가
   - 폰트: 앱과 동일한 폰트 사용 권장
